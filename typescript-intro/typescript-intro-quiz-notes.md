# typescript-intro-quiz-notes

## Quiz Questions

After completing this exercise, you should be able to discuss or answer the following questions:

- When was TypeScript introduced?
  2012

- Who created TypeScript?
  Microsoft

- How can you compile TypeScript to JavaScript?
  Using tsc (TypeScript compiler)

- What are some benefits of using TypeScript?
  It reduces the number of runtime errors that JavaScript is prone to. Its main advantages are the tools it provides for writing more robust JavaScript. Especially as your projects grow in scale and complexity, TypeScript helps you catch bugs and errors faster, while also forcing the rest of your team to maintain a nice structured way of writing code.
